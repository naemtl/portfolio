MyMediaPlayer

Developed by Matthew Stinis and Sebastien Scullion

The purpose of this project was to create a functional design of a professional Windows based media player.

We took inspiration from industry-standard media players such as VLC Media Player, Windows Media Player, and iTunes.

Our goals were to create a player with a built in media library, which would enable the user to upload/download media from a database of media files unique to each user account, as well as create savable playlists.

Technologies used

  • WPF/XAML/C#
  • SQL Database
